What is inpaint-web?

inpaint-web is a free and open-source tool based on WebGPU and WASM technology that enables image restoration (inpainting) and upscaling directly in the browser. This means you don’t need to install any software; just open your browser to perform high-quality image processing.

Key Features

  • Inpaint (Image Restoration): Whether it’s blemishes or unwanted objects in your photos, inpaint-web can intelligently fill or remove them, making your images look flawless.

Technical Support

  • WebGPU: The next-generation Web graphics API provides more efficient graphics rendering capabilities.
  • WASM (WebAssembly): A binary instruction format designed for modern web browsers, allowing near-native performance in the browser.
