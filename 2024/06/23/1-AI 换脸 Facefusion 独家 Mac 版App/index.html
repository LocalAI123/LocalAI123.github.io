Facefusion: AI Face Swap App for Mac

Facefusion is an open-source AI video/picture face swapping tool. It comes with a brand new user interface and offers a web page operation similar to Stable Diffusion, greatly enhancing the user experience. The face fusion feature of this tool can precisely infuse the characteristics of the target face into the template image, generating a new image that contains the target facial features while maintaining the original posture, expression, and movement. Facefusion supports various fusion modes, including single face fusion, multi-face fusion, and specific face selection fusion.

Installation Guide

To simplify the installation process, Facefusion is packaged into a standalone launcher. Users can run it with a simple click, without the need to configure a complex Python environment.

Note: The Facefusion App only supports devices equipped with the Mac M1/2/3 series chips.
